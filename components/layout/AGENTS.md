# Layout Rules

- Reuse `PageShell`, `SiteHeader`, and `SiteFooter` for page scaffolding.
- Keep navigation/footer content sourced from `content/navigation.ts` and `content/footer.ts`.
- Use semantic button/text/border classes from `theme/semantic.ts`.
- Apply reusable typography classes (`typo-*`) for headings, labels, and body text.
- Avoid duplicating header/footer markup in page files.
